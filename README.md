Codename App
==============

Just-for-fun web API for [codename](https://github.com/rjz/codename)

### Installation

Pull it, run it.

    $ git clone git@github.com:rjz/codename-app
    $ cd codename-app && node index

### Tests

Just a few:

    $ npm test
